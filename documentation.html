Interpreter

The Vindicator is currently on version 0.1. New realeses are undefined and uncertain.

The Vindicator Mathematical Language is a language design with prefix notation, meant to be used only for mathematical operations. It's not a programming language.

The Vindicator accepts two data type: Number and Vector. Those two don't operate together (yet).
Both can be operated on two different types of operations: Unary and Binary. Unary accepts only 1 argumento, Binary expects 2.
For vectors, unary operations are applied element-wise, creating a new vector. While binary operations are executed on every pair of elements, from first to last, returning a single element as a result.
Operations on vectors are the same as on numbers, but are identified with a '$' at the front.

-----------------------------------------

# Numerical Examples
1. Unary Operation => sqrt 9
2. Binary Operation => + 1 2
3. Setting variable => set nice 69

# Vector Examples
1. Setting vector => vec nums 10 11 12 13
2. Binary Operation => $+ nums
3. Unary Operation => $sqrt nums

------------------------------------------

>> Operations Available:
